# DisCal Website

The official DisCal Bot website, built with [Next.js](https://nextjs.org/).

## Development

Requirements:

- [`yarn`](https://yarnpkg.com/en/) for package management
- Clone repository
    - `git clone git@github.com:DreamExposure/discal-web.git`
- Install dependencies
    - `yarn install`
- Build and develop
    1. `yarn clean`
    2. `yarn generate`
    3. `yarn dev`
